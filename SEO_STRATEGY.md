# 🔍 SEO Strategy for Wedding Vendor Directory

## Goal: Rank #1 for Wedding Vendor Keywords in Your City

### Phase 1: Foundation (Weeks 1-4)

#### Technical SEO
- ✅ Schema markup (LocalBusiness for each vendor)
- ✅ Meta tags (title, description for each page)
- ✅ Mobile responsive (tested)
- ✅ Fast loading (Core Web Vitals)
- ✅ Sitemap & robots.txt
- ✅ Structured data for reviews

#### Content SEO
- Create city landing pages (Bangalore, Mumbai, Delhi)
- Create category pages (Venues, Catering, Photography)
- Each vendor = unique indexed page
- Each review = unique content

### Phase 2: Build Authority (Weeks 5-8)

#### On-Site
- Write wedding guides & tips
- Vendor comparison posts
- Budget planning articles
- Wedding checklist
- Vendor spotlights

#### Off-Site
- Get featured in wedding blogs
- Partner with wedding planners
- Guest posts on relevant sites
- Build backlinks from vendor sites

### Phase 3: Growth (Weeks 9+)

#### Organic Traffic
- Monitor Search Console
- Track rankings for target keywords
- Improve content for low-rank keywords
- Build internal linking strategy

#### User Signals
- Increase reviews (better rankings)
- Improve user engagement
- Lower bounce rate
- Increase time on site

## Implementation

### 1. Optimize Each Vendor Page

```
Title: "The Grand Palace - Luxury Banquet Hall in Bangalore"
Meta: "₹3L-10L | 500+ capacity | 4.8★ Reviews"
Schema: LocalBusiness markup
```

### 2. Create City Landing Pages

```
/vendors?city=bangalore
Title: "500+ Wedding Vendors in Bangalore"
Meta: "Venues, Caterers, Photographers, Decorators"
Content: City-specific information
```

### 3. Create Category Pages

```
/vendors?category=venues
Title: "Wedding Venues Across India"
Meta: "1000+ venues with reviews, ratings, pricing"
Content: Category insights, tips
```

### 4. Build Internal Links

```
Homepage → City pages → Vendor pages → Reviews
```

### 5. Create Linkable Content

Blog posts that other sites link to:
- "2024 Wedding Trends"
- "Wedding Budget Guide"
- "Vendor Selection Checklist"
- "Top Wedding Venues"
