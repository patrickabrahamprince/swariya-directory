## ⚡ 5-Minute Quick Start

### 1. Setup Locally
```bash
npm install
```

### 2. Create .env.local
Copy `.env.local.example` → `.env.local`

### 3. Run
```bash
npm run dev
```

### 4. Visit
http://localhost:3000

✅ You should see homepage with search bar!

## 🆘 Common Questions

**Q: Do I need to pay for anything?**
A: No! Everything is free:
- Vercel hosting (free)
- Supabase database (free)
- GitHub (free)
- Deployment (free)

**Q: How long to setup?**
A: 2 hours total:
- Local setup: 15 min
- Deploy: 10 min
- Database: 20 min
- Add vendors: 1 hour

**Q: Can I integrate with my Framer site?**
A: Yes! Add a "Browse Vendors" link that goes to your new site.

**Q: How do I get vendor data?**
A: Three options:
1. Manually add via admin panel
2. Use Crawl.AI to scrape (free tier)
3. Email outreach to vendors

**Q: What if I want to add payments?**
A: Add Razorpay integration (₹0 setup cost)
